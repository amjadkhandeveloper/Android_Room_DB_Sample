# Android Room Database Sample

A sample Android application demonstrating Room Database usage with Kotlin.

## Features

- Room Database integration
- Type converters for Date objects
- LiveData observables
- Coroutines for database operations
- Jetpack Compose UI

## Technologies Used

- Kotlin
- Room Database
- Jetpack Compose
- Coroutines
- LiveData

## Setup

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle files
4. Run the app
